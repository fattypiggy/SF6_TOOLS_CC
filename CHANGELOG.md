# Changelog

## Unreleased

## 0.9a - 2026-06-29

### Fixed

- 共享热键绑定支持键盘与 HID/GamePad 设备按钮，手柄、Hitbox、摇杆等可被游戏识别的设备可自由绑定训练快捷键。
- 修复设备按键点按绑定无反应的问题，并增加游戏输入流兜底与热键菜单设备检测显示。
- 保留旧版键盘热键配置兼容，已保存的 `{ vk, mods }` 绑定仍可继续使用。

## 0.9 - 2026-06-28

### Major

- 将训练运行时的帧稳定性作为重点优化目标，降低开启 MOD 后的细微掉帧和体感卡顿。
- 移除连段训练中每 60 帧全量扫描并解析连段 JSON 的轮询逻辑，避免连段文件数量较多时产生周期性卡顿。

### Changed

- RuntimeSafety 训练上下文检查改为每帧缓存一次，减少多个模块重复扫描 TrainingManager 和原生 UI widget。
- 训练管理器的 inactive WebState 写入改为状态变化优先、低频刷新，避免非训练状态下每帧同步写文件。
- 原生训练 UI 可见性扫描改为状态变化或低频刷新，不再每帧遍历并写入 widget 可见性。
- Distance Viewer 的 AA 招式列表按对手角色和数据版本缓存，不再每帧重建排序。
- Shared input hook 热路径改为数字栈并避免重复 finalizer 注册。

## 0.8e - 2026-06-26

### Fixed

- 修复嘉米连段中瞬狱杀指令显示错误。
- 修复训练快捷键拆分后的注册与调用问题。
- 调整停止训练操作的文字说明为“返回”。

### Changed

- 将连段训练 JSON 文件处理逻辑拆分为独立模块。
- 将连段训练、确认训练、距离显示的快捷键逻辑拆分为独立模块。
- 使用共享快捷键注册与配置模块统一管理训练功能快捷键。

## 0.8c - 2026-06-25

### Fixed

- 修复连段验证中当前攻击已在动作帧计入 `combo_cnt` 时，被错误判定为上一动作连击数不匹配的问题。
- 修复豪鬼 `2LP -> 5LK` 连段在第 3 步触发 `WRONG MOVE`，导致玩家复现和自动演示无法继续的问题。
- 修复瞬狱杀只显示技能名称、未显示 `LP LP 6 LK HP` 正确指令的问题。
- 恢复连段失败现场写入 `LastFail.json`，记录动作 ID、步骤、实时连击数、期望连击数和 HP。
- 修复距离显示和碰撞框在角色离开训练场中心后按双倍位移漂移的问题；碰撞偏移已是世界坐标，不再重复叠加角色位置。

### Added

- 中文化界面与术语替换
- 汉化连段指令附加文本，包括“强化”“瞬狱杀”“奖牌”“等级”“完美”“延迟”等。
- 中文字体支持
- 中文 Guides
- 训练 JSON 元数据字段 `_xt_meta`
- 默认作者设置
- 保存训练时填写名称、备注、作者、标签
- 网页训练管理器
- 本地训练扫描
- 网络 Manifest 训练库
- 训练安装 / 删除 / 更新
- `sync_signal.json` 无感刷新机制
- 项目来源、原作者署名与二次开发说明文档
- 连段训练场景状态录制与恢复能力调研文档

### Changed

- Trial 统一翻译为“训练”
- Combo Trial 统一翻译为“连段训练”
- 页面项目名称统一为“小吞街霸6训练MOD”
