# About

小吞街霸6训练MOD 是面向中文街霸6玩家的训练工具整合与增强项目。

它基于 WTT / SF6_Tools 的强大训练能力，进一步提供中文化、训练备注、作者信息、网页训练管理器、网络训练库等功能，让普通玩家更容易安装、查找、管理和分享连段训练。

本项目是基于 WTT 的中文增强版 / 训练平台扩展版，不宣称 WTT 原始代码为原创。

原作者：Wael3rd  
原项目：WTT / SF6_Tools  
原仓库：https://github.com/Wael3rd/SF6_Tools

基于 WTT / SF6_Tools 深度二次开发。本项目新增中文化、网页管理器与训练库功能。
