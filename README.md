# 小吞街霸6训练MOD

小吞街霸6训练MOD 基于 WTT / SF6_Tools 进行深度二次开发，是面向中文街霸6玩家的中文增强版 / 训练平台扩展版。

本项目不是宣称从零原创。本项目保留并尊重原作者署名与授权声明，发布修改版时应继续保留原作者 Wael3rd 的署名与原项目来源说明。

Original Project: WTT / SF6_Tools  
Original Author: Wael3rd  
Original Repository: https://github.com/Wael3rd/SF6_Tools

## 二次开发内容

在 WTT 原有训练工具能力基础上，本项目重点新增和维护：

- 中文化界面
- 中文字体与中文 Guides
- 连段训练元数据系统
- 训练名称 / 备注 / 作者 / 标签
- 网页训练管理器
- 本地训练 JSON 管理
- 网络训练库 Manifest
- 浏览器目录授权安装/删除/更新训练
- 训练库同步信号机制
- 面向中文玩家的使用流程优化
